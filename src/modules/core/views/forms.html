Forms are the main way you collect info from the user. They register, send you feedback, and talk between themselves thanks to forms. So here you have a list of form components you can use with Material CSS.

Text inputs

Text inputs allows the user to input some text data, as names, emails, phone numbers, addresses, etc.

To use them, just place an element with the "mc-input" class. Then add two children: The input, and the "mc-label".

  • <div class="mc-input" />
    • <input type="text | email | tel | number" />
    • <span class="mc-label"></span>
  • </div>



You can color the inputs using the "mc-bg-darkpurple" class on the "mc-input" element

  • <div class="mc-input mc-bg-darkpurple" />
    • <input type="text | email | tel | number" />
    • <span class="mc-label"></span>
  • </div>



Floating labels

You can also make the label be above the input always. Just place the "mc-floating-label" class on the "mc-label" element.

  • <div class="mc-input mc-bg-darkpurple" />
    • <input type="text | email | phone | number" />
    • <span class="mc-label mc-floating-label"></span>
  • </div>



Multi-line

If you want a multi-line input, just place a textarea instead an input, with the same "mc-input" class and the same children.

  • <div class="mc-input mc-bg-darkpurple" />
    • <textarea></textarea>
    • <span class="mc-label"></span>
  • </div>



Messages

In cases you want to display message, you can do it adding another children to the "mc-input", with the class "mc-input-message".

  • <div class="mc-input mc-bg-darkpurple" />
    • <input type="text | email | tel | number" />
    • <span class="mc-label"></span>
    • <span class="mc-input-message"></span>
  • </div>



Errors

In cases you want to display an error message, you can do it adding another children to the "mc-input", with the class "mc-input-error".

  • <div class="mc-input mc-bg-darkpurple" />
    • <input type="text | email | tel | number" />
    • <span class="mc-label"></span>
    • <span class="mc-input-error"></span>
  • </div>



Dropdowns

Dropdowns allow you to choose one option between more than one. They are used to display questions with more than 4 options and only one possible choice.

To use them, just place an element with the "mc-dropdown" class. Now you've the container, you only need to add 3 elements into it: the select, the "mc-dropdown-bg" with a "mc-dropdown-value" into it, and a "mc-dropdown-menu" with the mc-layout="list" on it.

  • <label class="mc-dropdown" >
    • <select>
      • <option value="val1">Text1</option>
      • <option value="val2">Text2</option>
    • </select>
    • <div class="mc-dropdown-bg" >
      • <div class="mc-dropdown-value" ></div>
    • </div>
    • <div class="mc-dropdown-menu" mc-layout="list"></div>
  • </label>

Radio Buttons

Radio Buttons allow you to choose one option between more than one. They are used to display questions with 2 or 3 options and only one possible choice.

To use them, just place a label with the "mc-radiobutton" class. Now you've the container, you only need to add 3 elements into it: the radiobutton, the "mc-radio-bg", and the "mc-label" with the text you want to display.

  • <label class="mc-radiobutton" >
    • <input type="radio" />
    • <div class="mc-radio-bg" >
      • <div class="mc-radio" ></div>
    • </div>
    • <span class="mc-label">Radio Button label</span>
  • </label>



You can color the radio buttons using the "mc-bg-darkpurple" class on the "mc-radio" element

  • <label class="mc-radiobutton mc-bg-darkpurple" >
    • <input type="radio" />
    • <div class="mc-radio-bg" >
      • <div class="mc-radio" ></div>
    • </div>
    • <span class="mc-label">Radio Button label</span>
  • </label>




Checkboxes

Checkboxes allow you to select multiple options in the same question. They are used to display questions with 2 or 3 options and more than one possible choice.

To use them, just place a label with the "mc-checkbox" class. Now you've the container, you only need to add 3 elements into it: the checkbox, "mc-checkbox-bg", and the "mc-label"with the text you want to display.

  • <label class="mc-checkbox" >
    • <input type="checkbox" />
    • <div class="mc-checkbox-bg" >
      • <div class="mc-checkbox-check" ></div>
    • </div>
    • <span class="mc-label">Checkbox label</span>
  • </label>



You can color the checkboxes using the "mc-bg-darkpurple" class on the "mc-checkbox" element

.
  • <label class="mc-checkbox mc-bg-darkpurple" >
    • <input type="checkbox" />
    • <div class="mc-checkbox-bg" >
      • <div class="mc-checkbox-check" ></div>
    • </div>
    • <span class="mc-label">Checkbox label</span>
  • </label>




Switchers

Switchers allow you to decide on|off questions. They are used to display configs settings.

To use them, just place a label with the "mc-switcher" class. Now you've the container, you only need to add 3 elements into it: the checkbox, the "mc-switch-bg", and the "mc-label" with the text you want to display.

  • <label class="mc-switcher" >
    • <input type="checkbox" />
    • <div class="mc-switch-bg" >
      • <div class="mc-switch" ></div>
    • </div>
    • <span class="mc-label">Switcher label</span>
  • </label>



Of course, you can color the switchers using the "mc-bg-darkpurple" class on the "mc-switch" element

.
  • <label class="mc-switcher mc-bg-darkpurple" >
    • <input type="checkbox" />
    • <div class="mc-switch-bg" >
      • <div class="mc-switch" ></div>
    • </div>
    • <span>Switcher label</span>
  • </label>




Sliders

Sliders are used to select a value in a range.

To use them, just place an element with the "mc-slider" class. Now you've the container, you only need to add a children with the slider (an input with "range" type), an element with the "mc-slider-bg" class, and it's chldren "mc-slider-progress" and "mc-slider-picker".

  • <div class="mc-slider" >
    • <input type="range" min="0" max="100" />
    • <div class="mc-slider-bg" >
      • <div class="mc-slider-progress"></div>
      • <div class="mc-slider-picker"></div>
    • </div>
  • </div>



Of course, you can color the progress bars using the "mc-bg-darkpurple" class on the "mc-slider" element

.
  • <div class="mc-slider mc-bg-darkpurple" >
    • <input type="range" min="0" max="100" />
    • <div class="mc-slider-bg" >
      • <div class="mc-slider-progress"></div>
      • <div class="mc-slider-picker"></div>
    • </div>
  • </div>




Progress Bar

Progress Bars are used to indicate the progress of something: Page load, video load, video timing, audio timing, etc.

To use them, just place an element with the "mc-progressbar" class. Now you've the container, you only need to add a children with the "mc-progress" class, and set the width in percentage format.

  • <div class="mc-progressbar" >
    • <div class="mc-progress" style="width: 20%;" ></div>
  • </div>



Of course, you can color the progress bars using the "mc-bg-darkpurple" class on the "mc-progressbar" element

.
  • <div class="mc-progressbar mc-bg-darkpurple" >
    • <div class="mc-progress" style="width: 20%;" ></div>
  • </div>




Spinners

Spinters are used to indicate an indeterminate load.

To use them, just place an element with the "mc-spinner" class. Now you've the container, you only need to add a children with the "mc-spinner-progress" class.

  • <div class="mc-spinner" ></div>



Of course, you can color the spinners using the "mc-color-darkpurple" class on the "mc-spinner" element

.
  • <div class="mc-spinner mc-color-darkpurple" ></div>




In case you have a particular background color on your section, you should use the "mc-bg-darkpurple" class on the "mc-spinner" element

.
  • <div class="mc-spinner mc-bg-darkpurple" ></div>



Disabled Elements

Of course, you can disable any of these elements by adding the "disabled" attribute, as you would do normally, and they'll look like this

  • <div class="mc-input" />
    • <input type="text | email | tel | number" disabled />
    • <span class="mc-label"></span>
  • </div>
