Navigation is an important part of your application. It allows the user to move between pages and see interesting content. Because of that, we have paid attention to this so our framework is completly responsive.

So, let's start! Just add an element with the "mc-navigation" class to create the navigation container. This container will hide automatically on mobile enviroments, and you'll be able to open it by clicking the "menu" button, or swiping to de right.

Navigation title

The menu bar contains the header bar on the navigation panel. It'll be hidden automatically on mobile enviroments and will appear when clicking the "Menu" button, as the navigation panel.

Simply add an element with "mc-header-menu" class into your the "mc-navigation".

You can place Icon Buttons and Titles into this bar.

  • <div class="mc-navigation">
    • <div class="mc-header-menu">
      • <span class="mc-button-icon" mc-action="menu"></span>
      • <span class="mc-button-icon" mc-action="back"></span>
      • <p class="mc-title"> Menu </p>
    • </div>
  • </div>

Account Info

The account info is an special header for the navigation panel to display some info about the logged user. It is very useful for identifying our account, and it's very beautiful also.

Simply add the "mc-accountinfo" class to your "mc-header-menu" element and it'll be ready to display your new components.

You can place 3 different elements on it:

  • An image
  • First line of text (recommended mc-title or mc-subtitle)
  • Second line of text (recommended mc-text or mc-note)

  • <div class="mc-navigation">
    • <div class="mc-header-menu mc-accountinfo">
      • <img />
      • <h1 class="mc-title"> User Name </h1>
      • <p class="mc-text"> user email </p>
    • </div>
  • </div>

Navigation items

These items are the main items, and they can be added by writing the "mc-nav-item" class.

In our menu, this corresponds to "Home", "Elements", "Colorize", etc...

You can add an icon to your item or subitem easily. Just add an "mc-action" attribute, like Icon Buttons

In our menu, main items have icons.


  • <div class="mc-navigation">
    • <a href="#" class="mc-nav-item" mc-action="home"> Home </a>
    • <a href="#" class="mc-nav-item" mc-action="text"> Texts </a>
    • <a href="#" class="mc-nav-item" mc-action="icon"> Icons </a>
    • ...
  • </div>

Navigation subitems

Subitems are in the second level, and they are below the main items. You can create a div to contain all your related subitems, and then, add all the elements with "mc-nav-subitem" class into it.

In our menu, this corresponds to "Texts", "Icons", "Navigation", etc. under "Elements" main item.


  • <div class="mc-navigation">
    • <a href="#" class="mc-nav-item" mc-action="home"> Navigation </a>
    • <div>
      • <a href="#" class="mc-nav-subitem"> Nav. Items </a>
      • <a href="#" class="mc-nav-subitem"> Nav. Subitems </a>
      • ...
    • </div>
  • </div>

Collapsable subitems

You can hide subitem by adding the "mc-collapsed" class to their parent container. You also must place an "mc-submenu" on it, with the same value that the "mc-menu" you will place on its related ".mc-nav-item"

  • <div class="mc-navigation">
    • <a href="#" class="mc-nav-item" mc-action="home" mc-menu="navigation"> Navigation </a>
    • <div class="mc-collapsed" mc-submenu="navigation">
      • <a href="#" class="mc-nav-subitem"> Nav. Items </a>
      • <a href="#" class="mc-nav-subitem"> Nav. Subitems </a>
      • ...
    • </div>
  • </div>
