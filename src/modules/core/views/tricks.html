We have implemented some tricks to improve the use of Material CSS, like some functions or minor elements.

With this tricks, we try to improve even more the creation of beautiful webapps, following Material Design guidelines.

Splash Screen

The splash screen is a perfect way to hide the page content while it is loading. This helps the user to hide the "this is too slow" feeling.

Simply add an element with "mc-splash" as your body's first child, and into it place an "mc-content" element.

Now you are prepeared to place your image, title, subtitle, text, note or maybe a spinner.

  • <div class="mc-splash">
    • <div class="mc-content">
      • <img src="" />
      • <div class="mc-spinner"></div>
    • </div>
  • </div>

Toogle Navigation

If you want to force the navigation panel to open, close or simply toogle between those states, you can use the "mc.toogleNavigation()" function:

  • mc.toogleNavigation( true ) // Force the navigation panel to open
  • mc.toogleNavigation( false ) // Force the navigation panel to close
  • mc.toogleNavigation() // Toogle between open/close

Toogle SubMenu

If you want to change the submenu's opened/closed status, you can use the "mc.toogleSubMenu()" function:

  • mc.toogleSubMenu( param ) // Change the open/close status, where param is the "mc-sumbenu" attribute's value

Toogle Tabs bar

If you want to force the tabs bar to open, close or simply toogle between those states, you can use the "mc.toogleTabs()" function:

  • mc.toogleTabs( true ) // Force the tabs bar to open
  • mc.toogleTabs( false ) // Force the tabs bar to close
  • mc.toogleTabs() // Toogle between open/close

Open/Close dialogs

If you want to open or close Dialogs (popups), you should use the "mc.openDialog()" and "mc.closeDialog()" functions:

  • mc.openDialog( "#myDialogId" ) // Force the dialog to open
  • mc.closeDialog( ".myDialogClass" ) // Force the dialog to close

Device type

If you want to know what type of device is displaying the app (so you can choose if your app do or not something), you have different functions to ask:

  • mc.isMobile() // Returns true if the screen fits with normal mobile sizes
  • mc.isTablet() // Returns true if the screen fits with normal tablet sizes
  • mc.isPC() // Returns true if the screen fits with normal PC sizes
  • mc.isTV() // Returns true if the screen fits with normal TV sizes

Also, we have different CSS classes for you to display the content on some particular devices. Of course, you can combine them as you want.

  • <img class="mc-mobile" /> <!-- Display the element only on mobile devices -->
  • <img class="mc-tablet" /> <!-- Display the element only on tablet devices -->
  • <img class="mc-pc" /> <!-- Display the element only on PC devices -->
  • <img class="mc-tv" /> <!-- Display the element only on TV devices -->
  • <img class="mc-mobile mc-tablet" /> <!-- Display the element only on mobile and tablet devices -->
