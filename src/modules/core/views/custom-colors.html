We also provide you a template with custom colors.

Instead of coloring the items with color classes (mc-color-blue, mc-bg-blue, etc...), you can use these custom colors to define custom colors and change them very quickly.

Main color

Main color has 3 classes dedicated to define its color behaiviour:

  • .mc-color-main: This class define the color of this item will be colored with the main color.
  • .mc-bg-main: This class define the background of this item will be colored with the main color.
  • .mc-bg-main:active: This class modifies the background of this item when clicking on it.

Secondary color

Secondary color has 3 classes dedicated to define its color behaiviour:

  • .mc-color-secondary: This class define the color of this item will be colored with the secondary color.
  • .mc-bg-secondary: This class define the background of this item will be colored with the secondary color.
  • .mc-bg-secondary:active: This class modifies the background of this item when clicking on it.
