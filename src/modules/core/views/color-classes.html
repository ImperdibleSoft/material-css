Coloring components it's really easy with Material CSS. There are 2 kind of classes for color your components.

Main color

Adding the class "mc-color-blue" the component will color its text and icons with blue. You can change "blue" for any color from the list.

  • <span class="mc-button mc-button-text mc-color-blue">Colored Button</span>

mc-color-black mc-color-green mc-color-blue mc-color-red

Background color

Adding the class "mc-bg-blue" the component will color its background with blue. You can change "blue" for any color from the list.

  • <span class="mc-button mc-button-text mc-bg-blue">Colored Button</span>

mc-bg-white mc-bg-green mc-bg-blue mc-bg-red
