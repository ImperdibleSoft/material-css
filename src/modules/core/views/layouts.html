Layouts are a great thing on Material CSS. You can use the "mc-layout" in any container element, so you can display 1, 2 or 3 colums. First of all, add the "mc-layout" class and "mc-layout" attribute. On the attribute, you can set the number of columns you want to display.

Once you have set this, you must create some children (they will be the columns), with the "mc-layout" class and the "mc-layout-size" attribute, where you can set the size of this column. Available sizes are:

  • Small
  • Normal
  • Big
  • Biggest

No layout

You can place all your content with no layout specified, so the content will addapt automatically to your screen.

  • <div class="mc-card">
    • ...
  • </div>

This layout will display cards in 1 column always, and they will resize themselves to look beautiful on any screensize.

    • <div class="mc-layout" mc-layout-size="small | normal | big | biggest">

Front page

This layout is one of the most shocking layouts availables on Material Design. It is used for landing pages and it's too simple to use.

Just add the attribute mc-layout="front" to your body, and see what happens.

  • <body mc-layout="front">
    • <div class="mc-navigation" >
      • ...
    • </div>
    • <div class="mc-header" >
      • ...
    • </div>
    • <div class="mc-content" >
      • ...
    • </div>
    • <div class="mc-floating" >
      • ...
    • </div>
  • </div>


Hidden Navigation

Maybe you want to hide the menu, but want a normal layout on your page. On these cases, you just have to add the attribute mc-layout="hidden-nav" to your body, and everything is going to be prepeared for that.

  • <body mc-layout="hidden-nav">
    • <div class="mc-navigation" >
      • ...
    • </div>
    • <div class="mc-header" >
      • ...
    • </div>
    • <div class="mc-content" >
      • ...
    • </div>
    • <div class="mc-floating" >
      • ...
    • </div>
  • </div>


No Navigation

Maybe we don't want a navigation menu on the left side. Maybe we want to display a menu on the tabs bar, or maybe we don't want a menu.

For this cases, we created this layout. Just add the attribute mc-layout="no-nav" to your body, and you are ready to go.

  • <body mc-layout="no-nav">
    • <div class="mc-header" >
      • ...
    • </div>
    • <div class="mc-content" >
      • ...
    • </div>
    • <div class="mc-floating" >
      • ...
    • </div>
  • </div>
