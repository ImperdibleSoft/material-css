Sometimes we need to show a lot of data in a reduced space. Tables are good for that, so we need to display them as beautiful as we can.

Adding the class "mc-table" to your table you'll be able to create your table normally, and it'll looks as great as this.

  • <table class="mc-table">
    • <tr>
      • <td> Data 1 </td>
      • <td> Data 2 </td>
      • <td> Data 3 </td>
      • <td> Data 4 </td>
    • </tr>
    • <tr>
      • ...
    • </tr>
  • </table>

Titles

An important element on tables are the titles. For this, we have given an special format to "th" elements, so you can use these elemens as usual.

First row

  • <table class="mc-table">
    • <tr>
      • <th> Title 1 </th>
      • <th> Title 2 </th>
      • <th> Title 3 </th>
      • <th> Title 4 </th>
    • </tr>
    • <tr>
      • ...
    • </tr>
  • </table>

Organization
