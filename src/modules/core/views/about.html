What is Material CSS?

Material CSS is a light CSS framework that allows you to create web apps with Material Design look&feel, and totally responsive.

This means you can create webs that will be displayed properly on Mobiles, Tablets, PCs and TVs, following the design guidelings proposed by Google for Android Apps.

When using Material CSS?

You can use Material CSS whenever you want.

If you want to create a website, a webapp, an extension for a browser... You can use Material Design on every development that uses CSS and JS.

Why Material CSS?

We have designed Material CSS in order to facilitate the creation of beautiful apps, so we can spend more time working on the app flow.

Another point that we have considered is to create homogeneous applications that are familiar to the user.

How does Material CSS work?

Material CSS has 2 files that must be used in order to work properly.

In one hand, we have the CSS file, that will set the styles and animations. In the other hand, e have a Javascript file, that will set some events to start these animations.

Optionally, there are some custom CSS themes that allows you to color your app like some famous applications.

Where can I use Material CSS?

Material CSS has been designed with the modern browsers in mind. So the apps you create with this framework will be compatible with:

  • Google Chrome
  • Mozilla Firefox,
  • Internet Explorer 11
  • Opera

On mobiles and tablets, it'll work on:

  • Google Chrome
  • Mozilla Firefox
  • Opera
  • Dolphin Browser
  • Safari
