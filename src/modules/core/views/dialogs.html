Dialogs are useful when you have to advice or ask some "yes/no" question to the user, so here we'll explain how make them look great.

Common dialog

You just need to add children to your body, with the "mc-dialog" class. This will prepare the dialog box with all the required styles.

  • <div class="mc-dialog">
    • <div class="mc-content">
      • <h1 class="mc-title"></h1>
      • <p class="mc-text"></p>
    • </div>
    • <div class="mc-footer">
      • <span class="mc-button mc-button-text mc-clickable">Close</span>
    • </div>
  • </div>

To open the popup, just place the "mc-opened" class on the mc-dialog and it'll be displayed, like this:

  • <div class="mc-dialog mc-opened">
    • ...
  • </div>

*We strongly recommend you to use the mc.openDialog(DOM selector) and mc.closeDialog(DOM selector) functions when open/close dialogs. This will fire the CSS animations in a proper order.

Falling dialogs

Adding the class "mc-falling-dialog" to the mc-dialog will simulate the dialogs fall from the top of the screen.

  • <div class="mc-dialog mc-falling-dialog">
    • <div class="mc-content">
      • <h1 class="mc-title"></h1>
      • <p class="mc-text"></p>
    • </div>
    • <div class="mc-footer">
      • <span class="mc-button mc-button-text mc-clickable">Accept</span>
      • <span class="mc-button mc-button-text mc-clickable">Close</span>
    • </div>
  • </div>


Bottom dialogs

Adding the class "mc-bottom-dialog" to the mc-dialog will make the dialogs appear from the bottom up.

In this case, we have added a new button to the footer, but it is too big, so we placed the "mc-complete-button" class on the mc-footer to make each button to fill all the width.

  • <div class="mc-dialog mc-bottom-dialog">
    • <div class="mc-content">
      • <h1 class="mc-title"></h1>
      • <p class="mc-text"></p>
    • </div>
    • <div class="mc-footer mc-complete-button">
      • <span class="mc-button mc-button-text mc-clickable">Let me think about it</span>
      • <span class="mc-button mc-button-text mc-clickable">Accept</span>
      • <span class="mc-button mc-button-text mc-clickable">Close</span>
    • </div>
  • </div>


Tooltips

Another way to display aditional information is using tooltips. For using them, you only need to add, just place an "mc-tooltip" next to the item you want to bind with.

To bind those elements, add the mc-tooltip="foo" attribute to the item, and the mc-item="foo" to the tooltip.

  • <span class="mc-button mc-button-icon" mc-action="send" mc-tooltip="sendButton"></span>
  • <div class="mc-tooltip" mc-item="sendButton">
    • <span>Here the explanation</span>
  • </div>
