Material CSS

Material CSS is a light framework that allows you to create webs with Material Design look&feel.

This is a responsive design framework, that will allow you to create websites prepeared for be displayed on mobiles, tablets, PCs and TVs.

Use it

If you want, you can use our hosted files.

CSS File

The main framework, that allows you to use special classes.

<link href="http://www.material-css.com/material-css.css" type="text/css" rel="stylesheet" />

JS File

Open/close the menu bar in movile enviroments and resize some images.

<script src="http://www.material-css.com/material-css.js" type="text/javascript"></script>

Install it

You can install our NPM module by running the next command.

npm install materialcss
