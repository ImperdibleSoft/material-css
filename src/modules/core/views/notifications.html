Notifications are the best way to inform the user about something, without taking care of what you are doing.

Common notification

First of all, you need to place a "mc-notification-container" class as your body's direct child. This will be a controlled place to stack all the notifications.

To open a notification, just create it by placing an element with the "mc-notification" class. The notification will appear automatically in its place.

  • <div class="mc-notification-container">
    • <div class="mc-notification">
      • ...
    • </div>
    • <div class="mc-notification">
      • ...
    • </div>
    • <div class="mc-notification">
      • ...
    • </div>
  • </div>

*We strongly recommend you to use the mc.updateFloatingButtons() function when open/close notifications. Thanks to this function, your notifications won't overlaps the floating buttons on mobile enviroments.


Notification components

You can add mc-title, mc-subtitle, mc-text, mc-note, mc-button-icon, mc-button-text and mc-button-comb elements to your notification, so the user can interact with them.
