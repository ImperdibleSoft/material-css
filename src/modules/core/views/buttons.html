We have some button types prepeared for implementing instantly. With the class "mc-button" we set the common styles for all buttons, so this is mandatory.

Icon Buttons

Adding the class "mc-button-icon" the button will be set as a round button, and it will be prepeared for implementing an icon into it.

Adding the attribute "mc-action" an icon will be placed into it. In this case, we have set mc-action="add".

  • <a class="mc-button mc-button-icon" mc-action="add"></a>


Text Buttons

Adding the class "mc-button-text" the button will be set as a rectangular button, that will resize automatically depending on the content.

  • <a class="mc-button mc-button-text">Delete</a>

Delete

Combined Buttons

Adding the class "mc-button-comb" the button will be set as a rectangular button, that will resize automatically depending on the content and will leave space for the icon specified on the "mc-action" attribute (like explained for Icon Buttons).

  • <a class="mc-button mc-button-comb" mc-action="share">Share</a>

share

Floating Buttons

Floating buttons are used to display some main functions on your app, like create an email, add a contact, etc...

If you want to add floating buttons, append to the "body" of your web an element with the "mc-floating" class. Then, just place some "Icon Buttons" on it, and they will be displayed properly.

  • <div class="mc-floating">
    • <a class="mc-button mc-button-icon" mc-action="scroll-top"></a>
    • <a class="mc-button mc-button-icon" mc-action="scroll-down"></a>
  • </div>

Button Effects

You also can make a button clickable, so you will see a deep effect when clicking the button. Just add the "mc-clickable" to the button.

  • <a class="mc-button mc-button-icon mc-clickable" mc-action="add"></a>
  • <a class="mc-button mc-button-text mc-clickable">Delete</a>
  • <a class="mc-button mc-button-comb mc-clickable" mc-action="share">Share</a>

Button Shadow

If you want buttons with shadow, just add the class "mc-button-raised" to the button.

  • <a class="mc-button mc-button-icon mc-clickable mc-raised" mc-action="add"></a>
  • <a class="mc-button mc-button-text mc-clickable mc-raised">Delete</a>
  • <a class="mc-button mc-button-comb mc-clickable mc-raised" mc-action="share">Share</a>
